# Branch: Custom UI Design

**Branch Objective**: Design and implement a title screen, pause menu, and custom HUD that matches your island's visual theme

**Why This Branch?**: Default Roblox UI works, but custom interfaces make your game feel complete and unique. UI skills transfer to any Roblox project.

---

- Mission B6.2.1 - ScreenGui Structure - Understand Frames, TextLabels, and ImageLabels as building blocks for interfaces
- Mission B6.2.2 - Title Screen - Create a welcoming start screen with your game's name and a "Play" button
- Mission B6.2.3 - Custom Timer Display - Replace boring text with a styled timer that fits your island theme
- Mission B6.2.4 - Mobile-Friendly Layout - Adjust UI positioning and sizes so your game looks great on phones and tablets
