# Branch: Debugging Like a Pro

**Branch Objective**: Master the Output window, print debugging, and common error patterns to fix scripts independently

**Why This Branch?**: Every programmer spends time debugging. Learning these skills early means you can solve problems yourself instead of getting stuck on confusing errors.

---

- Mission B4.1.1 - Reading Error Messages - Decode Roblox error messages to find the exact line and type of problem in your scripts
- Mission B4.1.2 - Strategic Print Statements - Use print() to track what your code is doing and confirm values at each step
- Mission B4.1.3 - Common Error Patterns - Recognize and fix the top 5 beginner mistakes (nil values, typos, wrong parent, missing connections)
