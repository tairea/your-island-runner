# Branch: Studio Interface Mastery

**Branch Objective**: Become fluent with Roblox Studio's essential tools, shortcuts, and view controls to build 3x faster

**Why This Branch?**: Many beginners struggle with the Studio interface, losing time on basic navigation. Mastering these tools early pays dividends throughout your entire project.

---

- Mission B1.1.1 - Camera Controls - Master orbit, pan, and zoom with mouse and keyboard shortcuts (F to focus, shift to speed)
- Mission B1.1.2 - Transform Tools - Use Move, Scale, and Rotate tools efficiently with grid snapping and precise inputs
- Mission B1.1.3 - Explorer Power User - Navigate the Explorer panel quickly using search, expand/collapse, and drag organization
