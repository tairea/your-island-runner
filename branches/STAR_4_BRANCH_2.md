# Branch: Variables and Data Storage

**Branch Objective**: Use variables and leaderstats to track player statistics like deaths, time, and checkpoints reached

**Why This Branch?**: Variables let your game remember things. Tracking player progress opens the door to leaderboards, achievements, and save systems.

---

- Mission B4.2.1 - Variable Basics - Create and modify variables to store numbers and text in your scripts
- Mission B4.2.2 - Leaderstats Setup - Display player statistics on the leaderboard using the leaderstats folder structure
- Mission B4.2.3 - Death Counter - Track and display how many times a player has respawned using IntValue objects
- Mission B4.2.4 - Best Time Tracking - Record and display the player's fastest completion time using variables
