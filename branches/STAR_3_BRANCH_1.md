# Branch: Terrain Sculpting Deep Dive

**Branch Objective**: Create natural-looking terrain with beaches, hills, and water features using advanced terrain editor techniques

**Why This Branch?**: Roblox's terrain system can create stunning natural environments, but most beginners only scratch the surface. These techniques make your island feel alive.

---

- Mission B3.1.1 - Terrain Materials - Experiment with all terrain materials (sand, grass, rock, mud) and learn when each looks best
- Mission B3.1.2 - Water Bodies - Create realistic water using terrain's water material, adjusting wave size and color properties
- Mission B3.1.3 - Elevation Variety - Use the Grow, Erode, and Smooth tools to create natural hills, cliffs, and valleys
- Mission B3.1.4 - Beach Transitions - Create smooth gradients between water, sand, and grass like a real shoreline
