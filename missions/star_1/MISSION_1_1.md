# Mission 1.1: Instant Setup

**Learning Objective**: Download and install Roblox Studio, open a baseplate project, and locate the Play/Stop buttons to test your game

## Summary
This mission gets you from zero to running Roblox Studio in the fastest possible way. You will create a Roblox account (if needed), download Roblox Studio, and open your first project using the Baseplate template. By the end, you will know how to press Play to test your game and Stop to return to editing.

## Key Concepts
- Roblox Studio is free software for creating Roblox games
- A Baseplate is a simple starting template with a flat ground and spawn point
- The Play button lets you test your game instantly
- The Stop button returns you to building mode

## Prerequisites
- A computer (Windows or Mac) with internet access
- Permission to install software (may need to ask your teacher or IT department)
- An email address to create a Roblox account
