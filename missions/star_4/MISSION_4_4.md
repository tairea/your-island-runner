# Mission 4.4: Death and Respawn

**Learning Objective**: Code kill bricks to return players to checkpoints

## Summary
Add challenge and consequence to your game by creating death zones! Learn how to script kill bricks that eliminate players when touched, automatically sending them back to their last checkpoint. This is essential for any obstacle course or platformer game.

## Key Concepts
- Humanoid.Health controls character life
- Setting Health to 0 kills the character
- Players automatically respawn at their RespawnLocation
- Kill zones add challenge and stakes to gameplay
- Visual design helps players identify danger

## Prerequisites
- Completion of Mission 4.3 (Checkpoint Spawns)
- Understanding of Touched events
- Working checkpoint system in your game
